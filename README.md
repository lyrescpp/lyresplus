# 🛡️ LYRES COURSE BOT
### Telegram-бот для подготовки разработчика БПЛА / Алмаз-Антей

---

## 🚀 Запуск за 10 минут

### Шаг 1 — Получи токен бота
1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя: например `LYRES Course`
4. Придумай username: например `lyres_cpp_bot`
5. **Сохрани токен** — выглядит так: `7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxx`

---

### Шаг 2 — Получи Groq API ключ (бесплатно)
1. Зайди на **https://console.groq.com**
2. Зарегистрируйся (можно через Google)
3. Слева — **API Keys** → **Create API Key**
4. **Сохрани ключ** — выглядит так: `gsk_xxxxxxxxxxxxxxxxxxxx`

> ✅ Groq полностью бесплатный для личного использования
> Модель: Llama 3.3 70B — очень мощная, отвечает по-русски

---

### Шаг 3 — Задеплой на Railway

1. Зайди на **https://railway.app** (GitHub аккаунт)
2. **New Project** → **Deploy from GitHub repo**
3. Загрузи папку с ботом в GitHub репозиторий
4. В Railway выбери этот репозиторий
5. Перейди в **Variables** и добавь:

```
BOT_TOKEN = твой_токен_от_BotFather
GROQ_API_KEY = твой_ключ_от_Groq
```

6. Railway сам установит зависимости и запустит бота

---

### Альтернатива: запуск локально

```bash
# Установи зависимости
pip install -r requirements.txt

# Создай файл .env (или задай переменные вручную в config.py)
# BOT_TOKEN = "..."
# GROQ_API_KEY = "..."

# Запуск
python bot.py
```
